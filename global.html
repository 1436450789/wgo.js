BoardObject

Represents object on the board. It can be stone, marker, or even empty field. If you specify property c it will express a stone, if you specify property type it will be handled as a marker. If you specify only coordinates, it will express an empty field. For markers you can add another properties, for example: text for marker LB (label).

Type:
  • Object
Properties:
Name Type Attributes Description
x number

X coordinate.
y number

Y coordinate.
c WGo.B | WGo.W <optional>

Color of the stone.
type string <optional>

Type of the marker.
